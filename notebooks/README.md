# Notebooks

This folder contains Jupyter/Colab notebooks used for data analysis and modeling.

## Files
- **ZOMATO.ipynb**  
  End-to-end analysis of the Zomato dataset including:
  - Data Cleaning & Preprocessing  
  - Exploratory Data Analysis (EDA)  
  - Visualizations  
  - Machine Learning Models (Logistic Regression, Decision Tree, XGBoost, etc.)  

## How to Run
1. Open the notebook in **Google Colab** or **Jupyter Notebook**.  
2. Make sure you have the dataset (stored in `data/` folder).  
3. Run the cells step by step.

# Plots

This folder contains all the visualizations generated during analysis.

## Files
- Saved plots from EDA and model evaluation (e.g., bar charts, correlation heatmaps, ROC curves, etc.)
- Images should be saved in `.png` or `.jpg` format for easy viewing.

## Notes
- Keep plot names descriptive, for example:
  - `avg_rating_by_location.png`
  - `roc_auc_xgboost.png`
  - `decision_tree_confusion_matrix.png`
- Organize plots subfolders if needed (e.g., `eda/`, `models/`).
